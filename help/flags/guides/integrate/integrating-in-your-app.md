# Intégration d’indicateurs dans l’application {#integrate}

Cette section fournit des instructions d’intégration pour tous les clients Flags, organisés par type d’application.

## Choisir le chemin d’intégration {#choose}

Sélectionnez le guide correspondant à votre type d’application :

1. [Guide de démarrage](startup-guide.md) — Commencez ici pour un aperçu général de toutes les étapes d’intégration
1. [Applications mobiles](mobile-applications.md) — Intégration de SDK pour les applications mobiles
1. [Applications web](web-applications.md) — Intégration de SDK pour les propriétés web

## Types d’applications en un coup d’œil {#overview}

| Type d’application | Intégration recommandée |
| --- | --- |
| **Application mobile** | AEP Mobile SDK — consultez les guides [Android](../sdk-releases/android/android-extension-integration-guide.md) et [iOS](../sdk-releases/ios/ios-extension-integration-guide.md) |
| **Application web** | Web SDK — voir [Guides d&#39;intégration Web](../sdk-releases/web/web-extension-integration-guide.md) |
