# Audience dans les indicateurs de fonctionnalité et les groupes de fonctionnalités {#audience-overview}

Indicateurs fournit un ciblage d’audience pour les indicateurs de fonctionnalité et les groupes de fonctionnalités. Vous pouvez combiner des variables contextuelles pour contrôler précisément quels utilisateurs reçoivent une fonctionnalité.

Pour le ciblage basé sur le contexte (par exemple, le ciblage par langue active ou type d’appareil de l’utilisateur ou de l’utilisatrice), consultez [Utilisation du contexte dans les règles d’audience](using-context-in-audience-rules.md).
