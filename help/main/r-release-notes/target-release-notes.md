# Notes de mise à jour de Target (préliminaires)

Cet article contient des informations préliminaires. Les dates de publication, fonctions et autres informations peuvent changer sans préavis.

**Dernière mise à jour : 6 juin 2022**

Pour consulter des informations sur la version actuelle, voir [Notes de mise à jour de Target](release-notes.md). Les informations sur ces pages peuvent être identiques selon le timing des versions. Les numéros de problème entre parenthèses sont destinés à une utilisation interne par [!DNL Adobe].

## [!DNL Target Standard/Premium] 22.6.1 (version échelonnée) ; 7-9 juin 2022)

Cette version sera disponible selon le calendrier échelonné suivant :

* **7 juin**: Région Asie-Pacifique (APAC)
* **8 juin**: Région des Amériques
* **9 juin**: Région Europe, Moyen-Orient et Afrique (EMEA)

Cette version comprend les améliorations et correctifs suivants :

* Utilisateurs avec la variable [!UICONTROL Éditeur] peut désormais modifier les audiences dans les activités actives. (TGT-43582)
* Un point d’exclamation ( ! ) n’est pas un caractère valide au début d’un nom d’audience. Un message s’affiche maintenant si un utilisateur tente d’enregistrer une audience avec un point d’exclamation comme premier caractère (par exemple !Londres). (TGT-43643)
* Correction d’un problème qui empêchait certains clients de modifier tous les attributs lors de la modification d’audiences dans la variable [!DNL Target] Interface utilisateur. Le [!UICONTROL Modifier l’audience] page affichait un attribut à modifier, même si l’audience avait plusieurs attributs. (TGT-43552)
* Correction d’un problème qui empêchait certains clients d’enregistrer les audiences combinées provoquées par la création de conteneurs &quot;vides&quot; par l’interface utilisateur de Target. (TGT-43588)
