# Recherche et filtrage de contenu

Recherchez des ressources par mots-clés dans la bibliothèque [!UICONTROL Offres] de [!DNL Adobe Target].

## Recherche d&#39;offres codées par mot-clé

1. Cliquez sur **[!UICONTROL Offres]** > **[!UICONTROL Offres de code]**.
1. Saisissez les mots-clés souhaités dans la zone **[!UICONTROL Rechercher dans]**.

   Vous pouvez utiliser la liste déroulante pour filtrer par [!UICONTROL Nom de l’offre], [!UICONTROL Chemins AEM] ou [!UICONTROL Balises AEM].

   Vous pouvez également filtrer à l’aide des options [!UICONTROL Type], [!UICONTROL Source] et [!UICONTROL Type AEM] en cliquant sur l’icône [!UICONTROL Afficher les filtres] ![Afficher l’icône Filtres](/help/main/assets/icons/Filter.svg) pour affiner davantage votre recherche.

## Recherche d&#39;offres d&#39;images par mot-clé {#section_2465A71BC95942588F586B1EC8B9E5DB}

1. Cliquez sur **[!UICONTROL Offres]** > **[!UICONTROL Offres Image]**.

1. (Facultatif) Basculez entre la vue [!UICONTROL Carte] et la vue [!UICONTROL Liste], cliquez sur l’icône [!UICONTROL Vue Carte] ( ![Icône de vue Carte](/help/main/assets/icons/ViewCard.svg) ) ou sur l’icône [!UICONTROL Vue Liste] ( ![Icône de vue Liste](/help/main/assets/icons/ViewList.svg) ) dans le coin supérieur droit de la bibliothèque [!UICONTROL Asset].
1. Cliquez sur l’icône **[!UICONTROL Contenu uniquement]** ( ![Icône Contenu uniquement](/help/main/assets/icons/RailLeft.svg) ) dans le coin supérieur gauche pour afficher la zone de recherche.
1. Dans la zone de recherche, saisissez un mot-clé pour les ressources à rechercher, puis appuyez sur [!UICONTROL Entrée].
1. (Conditionnel) Utilisez le volet **[!UICONTROL Filtres et options]** sur le côté gauche pour affiner votre recherche.

   Vous pouvez filtrer par :

   * **[!UICONTROL Type de fichier]** :
      * [!UICONTROL Images]
      * [!UICONTROL Documents]
      * [!UICONTROL Multimédia]
      * [!UICONTROL Archives]
   * **[!UICONTROL Taille du fichier]** : utilisez le curseur pour sélectionner la taille de fichier souhaitée : [!UICONTROL La plus petite], [!UICONTROL Petite], [!UICONTROL Medium], [!UICONTROL Grande] ou [!UICONTROL La plus grande].
   * **[!UICONTROL Dernière modification]** : utilisez le curseur pour sélectionner la période : [!UICONTROL Récent], [!UICONTROL Heure], [!UICONTROL Jour], [!UICONTROL Semaine], [!UICONTROL Mois], [!UICONTROL Année] ou [!UICONTROL All Assets].
   * **[!UICONTROL Statut d’approbation]** : [!UICONTROL Approuvé] ou [!UICONTROL Rejeté]
   * **[!UICONTROL Statut d’expiration]** : [!UICONTROL Expiré].
   * **[!UICONTROL Note moyenne]** : 1 à 5.
   * **[!UICONTROL Orientation]** : [!UICONTROL Horizontal], [!UICONTROL Vertical] ou [!UICONTROL Carré].
   * **[!UICONTROL Style]** : [!UICONTROL Couleur] ou [!UICONTROL Noir et blanc].
   * **[!UICONTROL Score d’utilisation]** : [!UICONTROL minimum] à [!UICONTROL maximum].
   * **[!UICONTROL Score d’impression]** : [!UICONTROL minimum] à [!UICONTROL maximum].
   * **[!UICONTROL Cliquez sur Score]** : [!UICONTROL Minimum] à [!UICONTROL Maximum].
